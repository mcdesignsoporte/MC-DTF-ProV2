# MC DTF Pro v3

Herramienta web Streamlit para preparar imágenes para DTF.

## Modos incluidos

- 📸 Quitar fondo de fotografía
- 🧼 Limpiar PNG transparente
- 🛡️ Conservar diseño completo
- ⬛ Quitar fondo negro
- 🖨️ Preparar para DTF
- 🌑 Diseño oscuro / quitar negro puro

## Recomendación para diseños como Mustang / Silverado

Usar: **🌑 Diseño oscuro / quitar negro puro** o **⬛ Quitar fondo negro**.

No uses IA para este tipo de diseño porque puede borrar letras, splash y efectos.
